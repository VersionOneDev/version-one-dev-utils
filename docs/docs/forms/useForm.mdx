# The useForms Hook

The `useForms` hook can be used to manage and validate form state.

## Configuration

A configuration object should be passed when calling the useForm hook.

| Property      | Value                                                                             |
| ------------- | --------------------------------------------------------------------------------- |
| mode          | The validation mode: `onBlur`, `onChange`, `onSubmit`, and `onTouched` (default). |
| schema        | Schema used to validate the form fields.                                          |
| defaultValues | Default values to pass to form fields.                                            |
| onSubmit      | A function called when the form is submitted.                                     |

## Connecting to the `<form>` component.

The `useForm` hook returns a `props` object that can be applied to a `form` element.

```jsx
import { useForm } from "version-one-dev-utils/forms";

export function Form() {
  const form = useForm({ ...config });
  return <form {...form.props} />;
}
```

## Registering Fields

```jsx
import { useForm } from "version-one-dev-utils/forms";

export function Form() {
  const form = useForm({ ...config });

  return (
    <form {...form.props}>
      {form.register("name", (field) => (
        <input {...field} />
      ))}
    </form>
  );
}
```
